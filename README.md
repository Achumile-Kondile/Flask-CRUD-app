# Flask CRUD App 🧑‍💻

This is a simple **Flask-based CRUD application** for managing users. You can create, read, update, and delete user data through a web interface.

---

## 🚀 Features

- 📝 Add new users  
- 📋 View list of users  
- ✏️ Update user details  
- ❌ Delete users
